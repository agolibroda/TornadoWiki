Список и назначение "рабочих столов" 

основным интерфейсом администрирования контента являются "рабочие столы" 
- это наборы списков ситеных сущностей, и инструментов реактироания

список:
- Персональный РС
- Групповой РС
- Администраорский РС

Доступ к РС Автор получает после регистрауии и авторизации


Персональный РС систоит из -
- списка всех статей, которые радактировал Автор 
- полного списка статей, которые есть в системе (публичных)
- списка групп, которые есть в системе (публичных и приватных)
- списка Авторов, зарегистрированных в истеме


соответсвенно, из списокв мохно перехти 
- на просмотр ревизий/редактирование статей
- к просмотру/редактированию  групп 
	(редактировать только те, в которых Автор является АДМИНОМ) 
	подать заявку на членство в группе
- к просмотру "витрин" Авторов


Витрна Автора
- Основные публичные данные
- список статей, которые редактировал Автор (только публичные)


Рабочий Стол Групы: 
- описание группы (название и аннотация)
- список статей в группе   
- список участников группы
- кнопка вступить в группу / удалится из группы

Права Редактирования
	Редактирование описания группы - только админы группы
	Создание новых статей в группе - все члены группы
	Редактирование статей в группе - все члены группы  

Видимость: 
Группа может иметь одну из двух уровней видимости:
	- публична группа (описание, список статей, список участников доступны любому пользователю)
	- приватная группа (описание доступно для просмотра любому пользователю  
			Списки статей и участников доступны только ленам группы)) 

Статьи в публичной группе могут быть публичными и групповыми 
	Публичные видны полностью (заголовок, аннотация, текст) любому наблюдателю 
	Групповые: соголовок и аннотация видны всем, текст - только участникам


	Подать заявку на участие (удалится) мохно нажатием на кнопку на "РС"		
	Добавить статью в группу - можно начав редактированиенового документа (кнопка в группе) 



статусы Автора в Группе:
- Претендент 		- The applicant
- Участник			- Member
- Админ				- Admin
- Удаленный 		- Remote
