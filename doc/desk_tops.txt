Концепция «рабочие столы»

Система разрабатывается для помощи Авторам в редактировании, обсуждении и оценке текстов по различным тематикам.
Одному Автору могут быть интересны разные тематики, для отображения текстов по различным тематикам вводится понятие «группа». Группа которая объединяет Авторов и их тексты. У группа есть название и описание.

В Проекте реализована концепция  «рабочие столы» 
- у каждого Автора есть собственный, персональный рабочий стол на этом столе Автор видит списки групп, в которых участвует Автор, и все свои материалы;
- Для каждой группы создается отдельный рабочий стол группы, на котором участники группы могут работать над текстами, относящимися к группе.

Список типов рабочих столов:
- РС Автора
- РС Участника группы
- РС Администратора группы
- РС Администратора системы

Доступ к РС Автор получает после регистрации и авторизации


РС Автора позволяет получить доступ к следующим инструментам:
- Списку всех статей, которые редактировал Автор; 
- Полному списка статей, которые есть в системе (публичных);
- Списку групп, которые есть в системе (публичных и приватных);
- Списку Авторов, зарегистрированных в системе;
- Форме редактирования новой статьи;

Функции Инструментов: 
«Список статей»  позволяет сортировать, фильтровать списки статей, передавать выбранные статьи инструменту 
«Форма редактирования статьи» или «Просмотр списка ревизий»
 «Список групп»  позволяет сортировать, фильтровать, просматривать описания групп, подавать заявки на вступление в группы, 
 создавать собственные группы.
«Список Авторов»  позволяет сортировать, фильтровать, редактировать, просматривать списки авторов, просматривать профили с 
помощью инструмента «Просмотр профиля Автора».
