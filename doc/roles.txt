
Система это: 
- Участники
- Статьи
- Инструменты ( 
-- Редактор статей, (по - идее, должен показывать, список групп, в которых содержится статья, если просматривает АВТОР статьи (или АС), то должно быть право менять флаги )
-- Список статей
-- Список Участников 
-- Список групп
-- Редактор прав (какие статьи в каких группах находятся, и с какими правами)
... 
)
и/или наполнение инструментов "опциями"

Инструменты собираются в "Инструментальные ящики" и раздаются пользователям в связи с их ролями.

Системные роли: 
- Участник  (У)
- АДМИН СИСТЕМЫ (АС)

Групповые роли:
- Участник Группы (УГ)
- Админ Группы (АГ)
- Редактор шаблонов (РШ) 

ПРАВА
Участник системы имеет право 
- Писать (редактировать) публичные статьи (pbl)
- Читать публичные статьи (pbl)
- Писать (редактировать) собственные статьи (sol)
- Читать собственные статьи (sol)
- Писать (редактировать) собственные статьи с разрешениями типа (grp)
- Читать собственные статьи с разрешениями типа (grp)

- создавать группы (при создании автор становится "АГ")
- подавать заявки на вступление в Группы
- передавать собственные и публичные статьи в группы (для собственных статей "sol" при этом меняется разрешение на "grp" )
	публичные статьи так и остаются публичными 

АГ
- Принимать в группу новых Участников
- Удалять Участников из Группы
- Назначать Участников "АГ"
- Добавлять новые статьи в Группу (авторские и публичные)
- Удалять статьи из Группы

УГ
- Писать (редактировать) статьи с разрешением (grp) (в тех группах, и те статьи, для которых есть право редактирования )
- Читать статьи с разрешением (grp) (в тех группах, и те статьи, для которых есть право чтения )
- Добавлять новые статьи в Группу (собственого авторства и публичные любого авторства) 
	(для собственных статей "sol" при этом меняется разрешение на "grp" )


роли и конфигурация инструментов:


АДМИН СИСТЕМЫ (инструментальный ящик):

!!! Редактор статей содержит возможности
 -- Выбор категории статьи (Служебная, Шаблон, Информационная) 
 -- Выбор шаблона (для категории "ИС")
 -- Выбор уровня разрешений 
 -- список групп, в которых находится редактируемая СТАТЬЯ (ля статьи уровня "grp") 
	(по - идее, должен показывать, список групп, в которых содержится статья, если просматривает АВТОР статьи (или АС), то должно быть право менять флаги )

!!! Список статей
-- Просто список с возможностью поиска/выбора по всему (паблик/групповому) набору статей (персоналки не трогать??) 
	и, надо как - то выделять (может цветом? статьи разных разрешений - мои личные  - одним цветом, групповые - другим, публичные  - третьим) 

!!! Список Участников 
-- Возможность создать Участника
-- Отредактировать данные Участника
-- Закрыть Участнику доступ
 
!!! Список Групп
 -- Возможность поиска, сортировки, фильтрации
 -- Редактирование Описаний Групп
 
!!! Редактор прав (какие статьи в каких группах находятся. )
 
 
 
!!! Профиль Группы (
 ---- Список Участников (возможность добавить/удалить Участника)
 ---- Список Статей (с флагами доступа)  с возможностью добавить/удалить статью, отредактировать флаг доступа

!!! Профиль Участника 
--- Общая информация
--- Список Групп, в которых Субъект состоит
--- Список Статей, которые написаны автором
