
Система это: 
- Участники
- Статьи
- Инсторументы ( 
-- Редактор статей, 
-- Список статей
-- Список Участников 
-- Список рупп
-- Редактор прав (какие статьи в каких группах нахоятся. )
... 
)
и/или наполнение инструментов "опциями"

Инструменты собираются в "Инструментальые ящики" и раздаются пользователям в связи с их ролями.

Системные роли: 
- Участник  (У)
- АДМИН СИСТЕМЫ (АС)

Групповые роли:
- Участник Группы (УГ)
- Админ Грппы (АГ)
- Редактор шаблонов (РШ) 

ПРАВА
Участник системы имеет право 
- Писать (редактировать) публичные статьи (pbl)
- Читать публичные статьи (pbl)
- Писать (редактировать)собственные статьи (sol)
- Читать собственные статьи (sol)
- Писать (редактировать)собственные статьи с разрешениями типа (grp)
- Читать собственные статьи с разрешениями типа (grp)

- создавать группы (при создании автор становится "АГ")
- подавать заявки на вступление в Группы

АГ
- Принимать в группу новых Участников
- Удалять Участников из Группы
- Назначать Участников "АГ"
- Добавлять новые статьи в Группу (авторские и публичные)
- Удалять статьи из Группы

УГ
- Писать (редактировать) статьи с разрешением (grp) (в тех группах, и те статьи, для которых есть право редактирования )
- Читать статьи с разрешением (grp) (в тех группах, и те статьи, для которых есть право чтения )
- Добавлять новые статьи в Группу (авторские и публичные)


роли и конфигурация инструметов:


АДМИН СИСТЕМЫ (инструментальный ящик):
 Редактор статей содержит возможности
 --- Выбор категории статьи (Служебная, Шаблон, Информацонная) 
 --- Выбор шаблона (для категории "ИС")
 --- Выбор уровня разрешений 
 --- список групп, в которых находится редактируемая СТАТЬЯ (ля статьи уровня "grp") 

Список статей
--- Просто сисок с возможностью поиска/выбора по всему (паблик/групповому) набору статей (персоналки не трогать??) 

Список Участников 
--- Возможность создать Участника
--- Отредактровать данные Участника
--- Закрыть Участнику доступ
....  
 
 Список Групп
 --- Возможность поиска, сортировки, фильтрации
 --- Редактирование Описанй Групп
 
 Редактор прав (какие статьи в каких группах нахоятся. )
 Просмотр содержимого Группы (
 ---- Список Участников (возможность добавить (удалить) Участника)
 ---- Список Статей (с флагами доступа)  с возможностью добавить/удалить отредактировать флаг доступа
 )
