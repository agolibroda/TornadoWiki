
Система это: 
- Участники
- Статьи
- Инсторументы ( 
-- Редактор статей, 
-- редактор прав
-- список статей
-- Список Участников 
... 
)

Инструменты собираются в "Инструментальые ящики" и раздаются пользователям в связи с их ролями.

Системные роли: 
- Участник  (У)
- АДМИН СИСТЕМЫ (АС)

Групповые роли:
- Участник Группы (УГ)
- Админ Грппы (АГ)
- Редактор шаблонов (РШ) 

ПРАВА
Участник системы имеет право 
- писать (редактировать) публичные статьи (pbl)
- читать публичные статьи (pbl)
- писать (редактировать)собственные статьи (sol)
- читать собственные статьи (sol)
- писать (редактировать)собственные статьи с разрешениями типа (grp)
- читать собственные статьи с разрешениями типа (grp)

- создавать группы (при создании автор становится "АГ")
- подавать заявки на вступление в Группы

АГ
- принимать в группу новых Участников
- Удалять Участников из Группы
- назначать Участников "АГ"
- добавлять новые статьи в Группу (авторские и публичные)
- удалять статьи из Группы

УГ
- писать (редактировать) статьи с разрешением (grp) (в тех группах, и те статьи, для которых есть право редактирования )
- читать статьи с разрешением (grp) (в тех группах, и те статьи, для которых есть право чтения )
- добавлять новые статьи в Группу (авторские и публичные)




    'pbl',
    'grp',
    'sol'
