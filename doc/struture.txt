Структура приложения:
сновной еденицей ЮИ в приложении  является 
"рабочий стол" - "РС" deskTop :-)

Инструменты
- редактор статьи - "ЕА"
- профиль пользователя - "ПП"
- список статей
- список пользователей
- список групп 

РС- 
- пользователя (одна позиция в меню) 
- участника Группы (Администратора) - для каждой группы, в которой пользователь учавствует
- Администратора системы одна штука (если есть роль)


Пользователь может иметь доступ к нескольким РС. (в зависимости от роли в данный момент)
в каждом РС есть некий набор инструментов. ВОзможности инструментов зависят от РС (и роли пользователя) 


нтерфейс пользователя:
- Персональный Рабочий Стол (РС) Учасника
- РС участника группы 
- РС Админа группы
- РС Админа ситемы 
- Редактор статей
 
 
 у статей ужен флаг - редактирует ТОЛЬКО АВТОР или, редактировать всем. (пдумать как правильно сказать.)
